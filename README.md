# 🛒 Procurement System (WPF)

## Description
Procurement System is a desktop application built with WPF (.NET) for managing procurement processes.  
It allows users to work with products, suppliers, and purchase orders in a simple and structured way.

The system includes authentication and role-based access control.

## Features
- User authentication  
- Role-based access (Admin, Procurement Specialist, Manager)  
- Product management  
- Supplier management  
- Purchase order creation  
- Search and filtering  

## Technologies
- C#
- WPF (.NET)
- MVVM
- Entity Framework Core
- SQL Server / SQLite

## Run
1. Open the solution in Visual Studio  
2. Configure database connection  
3. Press Ctrl + F5
